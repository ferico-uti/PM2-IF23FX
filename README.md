## Instalasi Node Modules

- API
  ```bash
    cd api
    npm i
  ```

- Mobile
  ```bash
    cd mobile
    npm i
  ```

## Development Mode
- Mobile
  ```bash
    npx expo install expo-dev-client
    # untuk instalasi Dev Client ke project
    # dilakukan sekali
  ```

  ```bash
    npx expo run:android
    # untuk menjalankan aplikasi dan build APK serta instalasi ke HP/Emulator
    # dilakukan bisa lebih dari sekali
  ```

  ```bash
    npx expo start --dev-client
    # untuk menjalankan aplikasi tanpa build APK
    # dilakukan bisa lebih dari sekali (jika APK sudah terinstal)
  ```
